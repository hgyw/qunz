    神武大帝诏曰：
    圣仁广运，凡天覆地载，莫不尊亲；帝命溥将，暨海隅日出，罔不率俾，贞珉大篆，荣施镇国之山。兹有慎勇侯夏建，讨伐西南十六国，为我朝廷开疆扩土，名在当世，功在千秋，今封其为九星侯爵。
